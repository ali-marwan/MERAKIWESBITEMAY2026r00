# Meraki Business Consultants

Premium UAE business consultancy and finance operations website.

**One system. One team. One standard.**

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript (strict)
- Tailwind CSS v3
- Manrope (via next/font)

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Deploy

Vercel-compatible. Connect this repository to a Vercel project — no environment variables required for v0.

## Roadmap (v1+)

- Stripe checkout for marketplace + packages
- Authentication on `/login` and `/portal/*`
- File upload + OCR for Meraki AI
- Live integrations to Zoho / QuickBooks / Xero / Tally
- Admin dashboard for client review queue
